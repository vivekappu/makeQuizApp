Welcome to MakeQuizApp

what is MakeQuizApp?

MakeQuizApp helps it's users to create Quizzes with various features and share it with other users .Teachers can create quiz and share it with student's , however there is no restriction that only teacher's should create quiz , anyone can create Quiz .

How to use ? Sign up You must sign up to create or attend quizzes create an account and start using the app! Creating Quizzes If you want to create quiz then read this

When you click create quiz from navbar you will see these three tabs
1.Quiz tab Here you can create quizzes 2.Question tab Here you can add question to your quizzes 3.Quiz report Here you can see the scores of quiz participants in the table and you can sort the list based on scores or name etc..

Attend Quiz If you want to attend quiz read this

To enter a quiz you need to have a valid passcode (secret code) set by the quiz creater which has been given to you . It's simple as that!!

Finally start Using Enjoy making and attending quizzes!! Hope you enjoy our services, we are happy to help you so contact us in case any problems occurs or to give us feedback
